Bokningssystem för Bio

En huvudmeny med menyval för att boka biljetter för en eller flera,
där priset på biljetteen skiljer på ålder.

Barn under 5 och pensionärer över 100 = Gratis
Ungdom under 20 || pensionär över 64 = Rabatterat

Menyval:
0. Avsluta
1. Lägg till person + Ålder
2. Visa alla personer med pris + totalpris

3. forloop övning, upprepa användarens input
4. string övning, ge tillbaka användarens tredje ord
